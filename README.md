# Custom URL Shortener 🔗

A fast, lightweight, and fully responsive custom URL shortener built with PHP and PDO. Designed with a clean, modern UI featuring a seamless dark/light mode toggle and a striking yellow accent. 

## ✨ Features

* **Mobile-Optimized UI:** A dashboard and login screen that adapt perfectly to mobile browsers with smart text-truncation to prevent horizontal scrolling.
* **Dynamic Theming:** Smooth transitions between Light (☀️) and Dark (🌙) modes, including automatic logo swapping and local storage memory.
* **Smart Dashboard:** Instantly filter and search through your shortened links without reloading the page.
* **One-Click Copy:** Easily copy your short links to the clipboard with visual confirmation.
* **Material Touches:** Enjoy subtle, premium UI details like Material 3 ripple effects on primary buttons.
* **Secure Admin Access:** Protected login system utilizing PHP sessions and secure password hashing.

## 🛠️ Tech Stack

* **Backend:** PHP (PDO for secure database interactions)
* **Database:** MySQL / MariaDB
* **Frontend:** HTML5, CSS3 (Custom Variables/Flexbox), Vanilla JavaScript
